Wave Maker

Introducing the Wave Maker, a dynamic p5.js project designed for web pages and logos. Powered by sine waves, this versatile tool enables users to customize the visual impact with adjustable parameters: lines, color palette, wave height, length, speed, and direction. Transform static elements into vibrant, animated waves, showcasing adaptability and creative control.
